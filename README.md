# Capstone Project: Energy Analysis

## Overview
This project analyzes energy production trends using data from the **BP Energy Outlook 2023**.  
All charts and dashboards were created in **Power BI** for demonstration purposes.

## Dashboards
- **Energy Demand Analysis**  
- **Transportation Energy Mix Analysis**  
- **Hydrogen Demand Forecast**  
- **EU Hydrogen Supply Sources**

## Skills Demonstrated
- Data analysis  
- Data visualization  
- Scenario comparison  
- Energy systems interpretation

## Tools
- Power BI Desktop  
- Power Query / DAX (optional, for transformations and calculations)  
- Excel (source of raw data)
